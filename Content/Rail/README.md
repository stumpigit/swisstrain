Materials and textures for the Swiss terrain railway system will be organized in this directory structure:

Tracks/
  - Materials for railway track visualization
  - Textures for different track types (standard gauge, etc.)
  - Spline visualization materials

Stations/
  - Materials for station buildings and platforms
  - Textures for different architectural styles
  - Station signage materials

Construction/
  - Materials for visualizing construction features
  - Bridge, tunnel, cut, and embankment visualization
  - Temporary preview geometry materials

Icons/
  - Editor mode icons
  - Tool button icons
  - UI elements for railway construction
