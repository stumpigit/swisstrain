# Swisstrain Terrain Prototype - System Requirements

## Unreal Engine Requirements
- Unreal Engine 5.5 installation
- Access to Epic Games Launcher or direct installation
- Minimum system requirements for Unreal Engine 5.5:
  - Windows 10/11 64-bit (as specified in the documentation)
  - Visual Studio 2019 or 2022
  - DirectX 11/12 compatible graphics card
  - 32 GB RAM minimum, 64 GB recommended

## Swisstopo Data Requirements
- Access to swisstopo digital height model (DHM) data
- Data format compatibility with Unreal Engine (likely require conversion to .raw or .png heightmap format)
- Coordinate system specifications (CH1903+/LV95 likely used by swisstopo)

## Development Tools
- Git with Git LFS for large file management
- Image processing software for heightmap conversion (if needed)
- Text editor or IDE for configuration files

## Target Prototype Area
No specific bounding box has been defined yet - this needs to be determined before proceeding with data acquisition and terrain creation.
